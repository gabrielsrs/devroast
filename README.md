# DevRoast

Drop your code. Get roasted.

DevRoast is a brutally honest code review app that gives your code the feedback it deserves — whether you want constructive criticism or full-on roast mode.

## Features

- **Paste & Roast**: Submit your code and get instant feedback
- **Roast Mode**: Toggle between helpful feedback and maximum sarcasm
- **Leaderboard**: See the worst code on the internet, ranked by shame
- **Syntax Highlighting**: Beautiful code display with line numbers

## How It Works

1. Paste your code into the editor
2. Toggle roast mode for extra sarcasm (or keep it helpful)
3. Hit submit and get roasted
4. Compare your score on the leaderboard

## Tech

Built with Next.js, Tailwind CSS, and a touch of sarcasm.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start roasting code.
